# TFG-Gasto-Electrico-de-Dogecoin
Codigo usado durante mi TFG "Calculo del Gasto Eléctrico de la Red de Dogecoin"
En este repositorio se incluye todo el codigo desarrollado durante el desarrollo del 
Trabajo Fin de Grado, además de los archivos necesarios para correrlo.

El quickshort es una implementación del algoritmo de ordenación, no fue diseñado por mi el algoritmo.

El resto de algoritmos y programas fueron diseñados por mi usando la API de Dogecoin.

El que calcula el gasto eléctrico es Calcular Gastar, mientras que los otros son para 
ver que mineros son los que minan Dogecoin.
